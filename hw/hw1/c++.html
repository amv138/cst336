   C++ Basics
                
    C++ is a high level language that is a derivative of C. C++ is considered 
    to be an improved version of C because of it's object oriented programming, 
    exception handling and many other ease of use features. It is a general use
    language meaning that is can be used for developing a wide range of 
    applications. 
    
    Features of C++ 
    1.) Better memory management, C++ allows you to dynamically
    allocate memory during runtime using the new and delete operator which 
    provides stronger data handling. 
    2.)Portability Most C++ compilers support ANSI standards which
    increases the portability of C++, meaning you can write the source code on one 
    operating system and run it on another without having to make changes. 
    3.) Object Oriented Unlike C, C++ has OOP features, meaning
    you have concepts such as Abstraction, Inheritance and Encapsulation which
    makes writing large scale programs much more efficient.
